# 💻 AI Codebase Assistant using Endee

## 🚀 Project Overview
This project is an AI-powered code search assistant built using a vector database approach inspired by Endee.

It allows users to search and understand codebases using natural language queries.

---

## 🧠 How It Works
1. Loads code files from a folder
2. Splits code into chunks
3. Converts code into embeddings using Sentence Transformers
4. Stores embeddings in a vector database (Endee-style)
5. Retrieves relevant code using semantic similarity

---

## 🔥 Features
- Semantic code search
- AI-powered code understanding
- Vector database (Endee-style)
- Fast similarity search
- Simple UI using Streamlit

---

## 🛠️ Tech Stack
- Python
- Streamlit
- Sentence Transformers
- NumPy
- Endee (Vector DB concept)
